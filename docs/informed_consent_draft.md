## セクション2: Litaについて（説明テキスト）

### Litaとは

Lita（LLM with Inner Thoughts Action）は、**自分から話しかけてくるAIチャットボット**の研究実装です。

一般的なAIチャットボットは「聞かれたら答える」反応型ですが、
Litaは内部で「思考」を生成し、適切と判断したタイミングで**自発的にメッセージを送信します**。

#### 利用プラットフォーム
- **Slack**（DMまたはチャンネル内でのメンション）

#### Litaの主な挙動
- あなたのメッセージに返答します（通常のチャットボットと同様）
- **約3分間やり取りがないと、Litaから話しかけてくることがあります**
- 会話の中からあなたの興味・関心を推定し、関連する情報を共有することがあります
- 絵文字リアクションで反応することがあります

## セクション3: 収集するデータ（説明テキスト）

### 収集されるデータについて

テスト期間中、以下のデータが収集・保存されます。

| データの種類 | 内容 | 保存場所 |
|---|---|---|
| **会話ログ** | あなたとLitaのすべてのメッセージ（発言日時を含む） | サーバー上のCSVファイル |
| **内部思考ログ** | Litaが生成した「心の声」とそのスコア（送信されなかったものも含む） | サーバー上のCSVファイル |
| **ユーザー記憶** | 会話から抽出されたあなたに関する情報（趣味、関心、仕事の話題など。最大100件） | メモリ上（セッション中） |
| **興味の推定結果** | 会話から推定されたあなたの興味・関心のリスト | メモリ上（セッション中） |
| **セッション統計** | 会話の長さ、ターン数、Litaの介入回数などの数値データ | サーバー上のCSV/JSONファイル |

#### 収集しないデータ
- Slack上のLitaとの会話以外のメッセージやチャンネルの内容
- あなたのSlackプロフィール情報（名前・アイコン等）を能動的に収集することはありません

## セクション4: データの利用目的（説明テキスト）

### データの利用目的

収集したデータは、以下の目的にのみ使用します。

1. **Litaの動作改善**: 会話の自然さ、介入タイミングの適切さの分析
2. **研究分析**: プロアクティブAIの有効性に関する定量・定性分析
3. **バグの特定・修正**: 不具合の調査と改善

#### データの取り扱い
- データにアクセスできるのは、開発・研究担当者のみです
- 個人が特定される形で社外に公開することはありません
- 研究発表等で利用する場合は、匿名化した上で使用します

## セクション5: あなたの権利（説明テキスト）

### テスト参加者の権利

テスト中、以下のスラッシュコマンドをSlackで使用できます。

| コマンド | できること |
|---|---|
| `/lita-memories` | Litaがあなたについて記憶している内容を確認できます |
| `/lita-interests` | Litaが推定したあなたの興味・関心を確認できます |
| `/lita-forget` | Litaの記憶をリセットできます |
| `/lita-export` | あなたの会話ログをエクスポートできます |
| `/lita-status` | 現在のセッション統計を確認できます |

#### その他の権利
- **中止の自由**: テストはいつでも中止できます。中止による不利益はありません
- **データ削除の請求**: テスト終了後、あなたのデータの削除を請求できます。開発担当者にご連絡ください
- **質問・懸念**: テストに関する質問や懸念がある場合は、いつでも開発担当者にご連絡ください

## セクション6: 注意事項（説明テキスト）

### ご注意いただきたいこと

- Litaは研究段階のAIです。不自然な発言や誤った情報を返すことがあります
- LitaはAIであり、専門的なアドバイス（医療、法律、金融等）を提供するものではありません
- Litaに共有した内容は記憶・記録されます。**機密情報や個人的にセンシティブな情報は共有しないでください**
- Litaが自発的に話しかけてくる頻度が気になる場合は、開発担当者にお知らせください
